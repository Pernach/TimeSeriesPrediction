# TimeSeriesPrediction

Предоставлен набор данных о заказах автомобилей. Для более точного распределения трудозатрат и рабочего времени, необходимо спрогнозировать количество заказов автомобилей на час вперед.

Задача: Значение метрики RMSE на тестовой выборке должно быть не больше 48.

Были проведены: ресемплинг, анализ данных, на основе которых добавлены новые признаки, позволяющие улучшить работу алгоритма.
Использованы библиотеки: pandas, numpy, statsmodels, matplotlib, seaborn, sklearn, catboost. Для решения задачи были использованы следующие алгоритмы машинного обучения: линейная регрессия, случайный лес и градиентный бустинг.

Приведены: построение модели, тестирование, полученные результаты приведены в виде сравнительной таблицы и на отдельном графике.
